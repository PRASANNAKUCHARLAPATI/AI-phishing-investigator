# Screenshots

This folder contains demonstration screenshots of the PHISHX phishing investigation workflow.

## Workflow Demonstration

The screenshots below show the end-to-end investigation process using the PHISHX interactive console.

### 1. Console Start
![Console Start](Screenshot%20from%202026-07-02%2011-42-49.png)

Launching the PHISHX interactive SOC console.

### 2. Load Email
![Load Email](Screenshot%20from%202026-07-02%2011-43-26.png)

Loading a sample `.eml` file for analysis.

### 3. Run Analysis
![Run Analysis](Screenshot%20from%202026-07-02%2011-44-55.png)

Executing the full investigation pipeline: parsing, IOC extraction, authentication checks, detection rules, threat intelligence, and report generation.

### 4. Investigation Summary
![Investigation Summary](Screenshot%20from%202026-07-02%2011-45-28.png)

Viewing the investigation summary with threat meter, verdict, score, and detection reasons.

### 5. Extracted Indicators
![Extracted Indicators](Screenshot%20from%202026-07-02%2011-46-04.png)

Listing extracted URLs, domains, IPs, and email addresses from the message.

### 6. Email Structure Details
![Email Structure](Screenshot%20from%202026-07-02%2011-46-47.png)

Drilling into headers, forms, and attachments for forensic detail.

### 7. Threat DNA & Campaigns
![Threat DNA](Screenshot%20from%202026-07-02%2011-47-19.png)

Viewing the threat DNA fingerprint, similar cases, and linked campaigns.

### 8. Detection Rules & Export
![Rules & Export](Screenshot%20from%202026-07-02%2011-47-34.png)

Generating Sigma, YARA, and Suricata rules from the investigation results.
